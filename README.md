# GraphQL server for Tickermaster

## Run locally
1. Set your Ticketmaster Api Token via `echo 'TICKETMASTER_API=yourKey' > .env`.
2. Start with `npm start`.

## tech stack
|name               |purpose|
|---                |---|
|Now                |Deployment|
|Apollo Server      |GraphQl Server based on Express|
|Typescript         |JS static typing|
